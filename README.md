# What Pressed - Community Atlases

Community-shared atlas repository for [What Pressed](https://github.com/made-by-gare/what-pressed).

Browse and install these atlases directly from the app via **Atlas Builder > Browse Community**.

## Submitting an Atlas

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide.

**Quick version:** fork this repo, add your atlas under `atlases/your-atlas-name/`, and open a PR. CI validates your submission automatically.

> **Note:** `index.json` is auto-generated by CI on merge. Do not edit it manually.
